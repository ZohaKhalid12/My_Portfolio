Hotel Management System
University Project - Programming Fundamentals.

Developed a hotel management system in C++ for my programming fundamentals course, featuring core functionalities like room booking, customer management, and billing.
I used vectors for efficient data storage, file handling for record keeping, and implemented loops and switches for a seamless, menu-driven interface.
This project strengthened my understanding of essential programming concepts and coding structures.

Book Recommendation System
University Project - Introduction to Data Science.

Developed a personalized book recommendation system using collaborative filtering.
Employed data preprocessing, exploratory data analysis (EDA), and feature engineering to refine the dataset.
Leveraged cosine similarity to find like-minded readers and suggest tailored book recommendations.
Enhanced user experience by analyzing user behavior and preferences.

Stocks' Prediction.
Self-Practice Project - ML & DL.

Developed a sophisticated stock price prediction model using LSTM neural networks.
Leveraged advanced statistical techniques to analyze historical stock data, uncovering hidden trends and seasonal patterns.
Visualized these patterns using time series plots and decomposition plots to gain deeper insights.
Fine-tuned the LSTM model to deliver accurate future price forecasts, empowering informed investment decisions.
